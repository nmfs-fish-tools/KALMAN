Kalman Filter
Kalman Filter (KALMAN)
Type: solver
author: Seaver, A.
The Kalman Filter is a recursive algorithm for updating linear projections of the system comprised by of an Observation Equation and a State Equation.Observation Equation: Kalman Observation Equation Image State Equation: Kalman State Equation The user will supply a vector of observations, coefficients of variation on observation error and scale vector (qt). The model will estimate two parameters: the state variable at time zero (b0) and the standard error on the state variable (sw). The model will return a vector of Filtered and Smoothed observations. Here is a technical manual available with details of equations used in the model.
